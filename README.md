# 筋トレ管理アプリ

## 機能一覧
- トレーニングメニューの一覧表示（ホーム画面）
- トレーニングメニューの作成
- トレーニングメニューの削除
- トレーニングメニューに種目を追加
- トレーニングメニューから種目を削除
- トレーニングメニューに登録している種目一覧の表示と更新
- 種目の作成
- 種目の削除
- 種目の一覧表示
- 公開されている種目の一覧表示
- パスワード変更(課題)

## 主な使用方法
1. トレーニングメニューを作成する（例: 「胸の日」、「背中の日」など）。
2. 種目をメニューに追加する。追加したい種目がなければ作成する。
3. トレーニングメニュー詳細画面から扱う重量を更新する。
